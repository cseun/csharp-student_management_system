# Student Management App

C# WinForms로 만든 간단한 학생 성적 관리 프로그램입니다.

## 기능
- 학생 성적 추가 / 수정 / 삭제
- 이름, 학년, 반, 번호 검색
- 시험 기준 석차 계산
- CSV 파일 저장 / 불러오기

## 개발 환경
- C#
- .NET WinForms
- Visual Studio
